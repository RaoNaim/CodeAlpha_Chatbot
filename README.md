# CodeAlpha Chatbot

This project is part of the CodeAlpha Cloud Computing Internship.

## Objective
To build an intelligent chatbot that handles multiple user queries and responds with relevant answers.

## Features
- Handles greetings, time, date, jokes, and basic questions
- Provides random intelligent responses for unrecognized input
- User can type `exit` or `bye` to stop the chat
- Simulates 500+ user queries through keyword detection

## Technologies Used
- Python
- Random responses
- DateTime module

## How to Run
1. Clone the repository
2. Open command prompt in the folder
3. Run: `python chatbot.py`
4. Chat with the bot by typing questions
5. Type `exit` or `bye` to stop
